HTML이란 무엇인가?

HTML:Hypertext Markup Language, 인터넷 서비스의 하나인 월드 와이드 웹을 통해 볼 수 있는 문서를 만들 때 사용하는 웹 언어의 한 종류이다. 특히 하이퍼텍스트를 작성하기 위해 개발되었으며, 인터넷에서 웹을 통해 접근되는 대부분의 웹 페이지들은 HTML로 작성된다.

HTML elements are the building blocks of html pages. With HTML constucts, images and other objects, such as interactive forms, as be embedded into the rendered page.
